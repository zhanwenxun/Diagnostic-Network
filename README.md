# Diagnostic-Network
基于机器学习的互联网医疗诊断APP

功能模块：
（1）个人信息模块
用户通过该模块完善个人信息，包括诊断所需要的基本信息以及联系方式、地址，满足用户直接进行医疗诊断以及急救；

（2）自主医疗模块
非医生用户通过该模块，填写最近出现的患病症状，软件的医疗诊断模型对其进行预测，给出最有可能患病结果，并给出建议的治疗方案；

（3）医生辅助医疗模块
医生用户可以通过该模块，连接本医院的数据库，对患者进行诊断，类似于上一模块；

（4）医院服务模块
用户可以通过该模块，查询当地医院、进行医疗预约、联系咨询专业医生，以便及时进行诊断和治疗；

（5）药物配送模块
用户可以通过该模块，购买自己所需要的药品、实时查询药物配送进度。
